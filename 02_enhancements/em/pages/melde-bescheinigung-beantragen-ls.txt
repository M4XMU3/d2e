Sie wohnen in München?
Oder Sie haben früher einmal in München gewohnt?
Dann können Sie beim Bürger-Büro
eine Melde-Bescheinigung beantragen.

Was ist eine Melde-Bescheinigung?

Eine Melde-Bescheinigung ist ein Dokument.
Eine Melde-Bescheinigung ist immer für eine Person.
In dem Dokument stehen Angaben zu der Person.
Die Angaben sind die Melde-Daten von der Person.
In der Melde-Bescheinigung stehen immer diese Informationen:

Ihr Nach-Name
Ihr Vor-Name
Ihr Geburts-Datum
Ihre Wohn-Adresse in München

Sie können selbst aussuchen:

Welche Angaben sollen noch auf der Melde-Bescheinigung stehen?

Zum Beispiel Ihre Staats-Angehörigkeit

Staats-Angehörigkeit bedeutet:
Dass Sie zu einem bestimmten Land gehören.
Zum Beispiel:

Deutschland
Oder Griechenland
Oder Türkei
Oder Ukraine

Zum Beispiel Ihre Religion

Religionen sind zum Beispiel:

Katholisch
Oder Evangelisch
Oder Jüdisch

Zum Beispiel Ihr Familien-Stand

Familien-Stand ist zum Beispiel:

verheiratet
Oder ledig
Oder geschieden
Oder verwitwet
Oder in eingetragener Lebens-Partnerschaft
Eine Lebens-Partnerschaft ist so etwas wie eine Ehe.
Es gibt die Lebens-Partnerschaft zwischen 2 Männern
oder zwischen 2 Frauen.

Zum Beispiel, ob Sie minderjährige Kinder haben

Minderjährig heißt:
Ihre Kinder sind noch nicht 18 Jahre alt.

Wofür brauchen Sie eine Melde-Bescheinigung?

Sie brauchen eine Melde-Bescheinigung zum Beispiel:

Bei der Bank, wenn Sie ein Bank-Konto haben möchten.
Wenn Sie einen Handy-Vertrag abschließen.
Für die Renten-Versicherung.
Wenn Sie heiraten.

Was kostet eine Melde-Bescheinigung?

Eine Melde-Bescheinigung kostet 5 Euro.
Eine Melde-Bescheinigung für die Renten-Versicherung ist kostenlos.

Wie bekommen Sie eine Melde-Bescheinigung?

Es gibt 3 Möglichkeiten.

Möglichkeit 1: Sie gehen in das Bürger-Büro in der Ruppert-Straße

Sie bekommen die Melde-Bescheinigung im Bürger-Büro in der Ruppert-Straße.
Die Adresse ist:
Ruppertstraße 19
80337 München
Sie müssen die Melde-Bescheinigung dort beantragen.
Sie müssen dafür selbst in das Bürger-Büro in der Ruppert-Straße gehen.
Sie brauchen vorher einen Termin im Bürger-Büro in der Ruppert-Straße.

So können Sie einen Termin im Bürger-Büro in der Ruppert-Straße machen

Sie können den Termin am Telefon ausmachen.
Das ist die Telefon-Nummer:0 89 - 23 39 60 00
Sie erreichen das Bürger-Büro amMontag, Dienstag, Mittwoch und Donnerstag:
Immer von 7.30 Uhr morgens bis 15.30 Uhr nachmittagsFreitag:
Von 7.30 Uhr morgens bis 13 Uhr mittags
Sie sprechen dann mit einem Mitarbeiter.
Der Mitarbeiter will wissen:

Sie brauchen eine Melde-Bescheinigung.
Ihren Nach-Namen
Ihren Vor-Namen
Ihre Adresse in München
Ihre Telefon-Nummer
Wann haben Sie Zeit für einen Termin bei dem Bürger-Büro?

Dann gibt Ihnen der Mitarbeiter von dem Bürger-Büro einen Termin.

Sie müssen diese Dinge zu dem Termin mitbringen

Sie müssen Ihren Personal-Ausweis oder Reise-Pass mitbringen.
Vielleicht haben Sie keinen deutschen Ausweis.
Dann müssen Sie uns den Pass von Ihrem Heimat-Land mitbringen.Wichtig
Eine Kopie von Ihrem Personal-Ausweis oder Pass reicht nicht.
Sie können das Antrags-Formular mitbringen.
Klicken Sie auf den Link zu dem Antrags-Formular.
Dann kommen Sie automatisch zu dem Antrags-Formular.
Drucken Sie das Antrags-Formular aus.
Bringen Sie das Antrags-Formular mit.Link zu dem Formular Antrag Melde-Bescheinigung

Achtung:
Das Formular ist in schwerer Sprache.
Können Sie nicht gut lesen oder schreiben?
Dann hilft Ihnen ein Mitarbeiter von dem Bürger-Büro beim Ausfüllen.
Sie müssen uns das sagen:
Welche Angaben sollen auf der Melde-Bescheinigung stehen.
Sie können uns das sagen:
Für welches Amt brauchen Sie die Melde-Bescheinigung.
Bringen Sie das Schreiben von der Behörde am besten mit.

Wie können Sie die Melde-Bescheinigung bezahlen?

Sie müssen die Melde-Bescheinigung im Bürger-Büro bezahlen.
Wenn Sie den Antrag bei einem Termin im Bürger-Büro stellen.
Sie können mit Bar-Geld bezahlen.
Das bedeutet:
Sie können mit Geld-Scheinen und Münzen bezahlen.
Sie können auch mit Ihrer Bank-Karte bezahlen.
Diese Karte haben Sie bei der Bank für Ihr Konto bekommen.
Sie können so bezahlen.
Und müssen keine Münzen oder Geld-Scheine dafür haben.
Sie können die Melde-Bescheinigung sofort mitnehmen.

Zum Beispiel Ihrem besten Freund.
